Discovering Credit Card Fraud

The purpose of this project was to analyze data to detect credit card fraud across accounts. In doing so, we built a program that could flag an account with fraud using machine learning.

After assembling the dataset, we used AWS to store the data, SQL to pull in the data, and python to code for machine learning. We also created the webpage and deployed it on Heroku.
